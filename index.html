📖 使用說明 Usage Guide
⚙️ 配裝流程
⚙️ Optimizer Flow
  • 工具內建角色資料，可直接使用，亦可自行匯入 CSV 更新
  • 匯入裝備 JSON，或手動輸入裝備資料
  • 調整套裝需求、速度區間及各屬性權重後，點擊開始配裝
  • 若無理想結果，可放低門檻，或使用遊戲內標準被動配裝
  • 儲存方案後可繼續配裝其他角色；另有「超級鎖定」可強制排除指定裝備
  • Character data is built-in; import an updated CSV anytime
  • Import equipment JSON or manually enter gear data
  • Set set requirements, speed range, and stat weights, then start the search
  • If results don't meet expectations, loosen requirements or use a standard passive build
  • Lock a plan and keep optimizing for other characters; use Super-Lock to hard-exclude specific pieces
🔧 裝備 JSON 匯出
🔧 Equipment JSON Export
  • 使用靈燕開發之瀏覽器外掛，可直接從遊戲頁面匯出裝備資料
  • 教學（非官方）：靈燕裝備匯出指南 →
  • 將匯出的 .json 檔案上傳至配裝工具右上角「裝備JSON」即可
  • 裝備資料與配置快取於本機，下次開啟無需重新匯入
  • Use the browser extension developed by 靈燕 to export gear directly from the game page
  • Guide (unofficial): 靈燕 Equipment Export Guide →
  • Upload the exported .json file via "Equipment JSON" in the optimizer's top-right
  • Data is cached locally — no re-upload needed on next visit
