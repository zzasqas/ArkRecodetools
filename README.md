## 對戰紀錄器

記錄每場 PvP 對戰結果，支援歷史搜尋與策略分析。

### 主要功能

- 記錄對手陣容、我方陣容、勝負與備註
- 支援角色暱稱搜尋（中英文皆可）
- 策略推薦：根據歷史資料分析常見對手組合
- 速度計算器（預設開啟）
- 可選連接自己的 Google Sheet 同步紀錄

### Google Sheet 同步（選用）

在「設定」頁面填入 Google Apps Script 網址，即可將紀錄同步至個人 Google Sheet。資料所有權完全由使用者掌控，本工具不儲存任何資料於外部伺服器。

## 角色資料庫

查詢角色基本資料、中英文名稱與暱稱對照，並可直接帶入推薦配裝設定。

### 主要功能

- 依屬性（火／水／木／光／暗）與職業篩選
- 顯示角色基礎數值、固有爆率爆傷
- 推薦配裝 Popup：點擊角色卡片查看推薦套裝與屬性門檻
- 點擊「套用並前往配裝」直接跳轉配裝計算器並帶入設定
- 參照煙雨閣強度表，後續若有更新將紀錄於更新紀錄

## 配裝計算器

匯入裝備後，依照套裝需求與屬性權重，自動搜尋最佳 8 組裝備組合。

### 裝備資料匯入

使用靈燕開發之瀏覽器外掛，從遊戲頁面直接匯出裝備 JSON。

教學（非官方）：https://ark-re-code-2f518a.gitlab.io/zh/guide/

匯出後於工具右上角點擊「裝備 JSON」上傳即可。資料會快取於瀏覽器 localStorage，下次開啟無需重新匯入。

### 配裝模式

**Mode A — 自由配置**

設定各屬性的下限門檻與優化權重（0～1），系統依加權分數排序。適合平衡型配裝或多目標探索。

**Mode B — 目標鎖速**

先設速度區間，再選擇要最大化的單一屬性。適合速度已確定、只想最大化特定屬性的情境。

### 主屬鎖定

在套裝需求旁可鎖定項鍊／戒指的主屬，避免非目標主屬裝備佔滿候選池：

- 💎 項鍊：爆率 / 爆傷 / 攻擊% / 生命% / 防禦% / 攻擊 / 生命 / 防禦
- 💍 戒指：命中 / 抗性 / 防禦% / 生命% / 攻擊% / 防禦 / 生命 / 攻擊

### 方案鎖定與超級鎖定

計算完成後，點擊「儲存方案」可存入 1～3 號方案。進入「🔒 鎖定/手動」Tab 查看已儲存的方案，並可繼續為其他角色配裝而不衝突。

超級鎖定（🔐）：在裝備庫中標記後，該裝備在所有角色的配裝中一律排除，適合確定要保留給特定角色的核心裝備。

### 手動配套

在「🖐 手動」分頁可手動選擇 6 件裝備，即時顯示與已儲存方案的數值差異（綠＝提升／紅＝下降）。

## 資料安全說明

- 所有資料儲存於瀏覽器 localStorage，不上傳至任何伺服器
- Google Sheet 同步為選用功能，使用者自行設定與管理
- 純前端架構，離線載入後配裝工具與角色資料庫均可離線使用
